# Security policy

## Reporting a vulnerability

If you find a security issue in this project, please **do not** open a public issue. Instead:

- Email: <azatkhairutdinov@gmail.com>
- Subject prefix: `[security] vision-recognizer`

You'll receive an acknowledgment within 7 days. Please allow up to 30 days for a fix before public disclosure.

## Supported versions

Latest `main` branch only. No backports to historical tags.

## Scope

This project is provided **as-is** under [MIT License](./LICENSE). No warranty, no SLA. Security reports are appreciated but not contractually obligated.
